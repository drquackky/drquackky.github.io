2Spaghi Pasta Bar's Customers Sentiment Analysis using Natural Language Processing Technique (NLP)¶

This datascience project is created for educational purpose to showcase the ability and my understanding in NLP technique. The data used in this object was obtained and granted permission to use by the owner of 2spaghi.

Pipeline for working¶

  • Combining 134 json files containing reviews into one dataframe

  • Data cleaning and transformation

    • clean data by dealing with unnecessary column and missing values
    • using pre trained model such as Vader from nltk library to classify the reviews into three categories: 'neutral', 'negative', 'positive'
    • double checking the misclassified class of reviews by manual checking

  • Data visualization (wordcloud, line chart, heatmap)

  • Create a pipeline to prepare the data for training model

    • remove punctuation
    • remove stopwords
    • perform count vectorization

  • Model training

    • split data to test-train
    • perform machine learning using Bayes Naive Classification, Logistic Regression

# Function to extract only the English translation
def extract_english_translation(row): 
    # Split by the translated label and take the second part
    parts = row.split('(Translated by Google)')
    if len(parts) > 1:
        # Clean and return the English translation
        return parts[1].strip().split('(Original)')[0].strip()
    return row.strip()  

# Apply the function to extract only English translations
df['english_reviews'] = df['reviews'].apply(extract_english_translation)

c. Using Vader Sentiment Analysing tool to classify sentiment¶

In [340]:
# create the new column called 'Sentiment'
# using trained model Vader to assign values
from nltk.sentiment.vader import SentimentIntensityAnalyzer
import nltk

# Download VADER lexicon
nltk.download('vader_lexicon')

# Initialize VADER
sia = SentimentIntensityAnalyzer()

# Function to categorize sentiment
def categorize_sentiment(review):
    scores = sia.polarity_scores(review)
    if scores['compound'] >= 0.05:
        return 'Positive'
    elif scores['compound'] <= -0.05:
        return 'Negative'
    else:
        return 'Neutral'

# Apply the function to the reviews column
df['sentiment'] = df['english_reviews'].apply(categorize_sentiment)

## create a pipeline
# 1. to remove punctuation
# 2. to remove stopwords
# 3. to perform count vectorization

def message_clean(message):
    text_punc_rm = [char for char in message if char not in string.punctuation]
    text_punc_rm_join = ''.join(text_punc_rm)
    text_punc_rm_join_sw_rm = [word for word in text_punc_rm_join.split() if word.lower() not in stopwords.words("english")]
    return text_punc_rm_join_sw_rm

  • After training using three techniques: Bayes Naive, Logistic regression. The one which works well for me seems to be Logistic Regression model.
  • The key point here is to distinguish between negative sentiments out from neutral and positive sentiments.
  • 17 Negative samples predicted as Negative (True Negatives for Negative class)
  • 1 Negative sample predicted as Positive (False Positives for Positive class).
  • This model is able to predict correctly negative class with precision 0.94 (94 percent were correctly classified). This indicates a high level of accuracy in predicting negative sentiments.
  • The sensitivity or recall of the model is 0.59, correctly identified 59%. This suggests that while the model is good at predicting negatives when it does, it misses 41% of the actual negative cases.
  • The model performs very well with positive sentiment but struggles with neutral sentiment, showing high precision but low recall.
  • While overall accuracy is high (84%), it may mask the underlying issues with the model's ability to detect neutral sentiment correctly.
  • The precision for the neutral class is perfect, but the recall is low, indicating that the model is likely biased towards predicting other classes
    (especially positive).
  • In order to have better performance, I plan to increase the number of negative and neutral samples to help balance the dataset
